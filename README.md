### Отладка консольного приложения «Фибоначчи»
### Использованные способы отладки
1. Точки останова (Breakpoints)
Использовались для остановки выполнения программы на определённой строке кода. Это позволило проверить текущее состояние приложения, значения переменных и найти место возникновения ошибки.

2. Пошаговое выполнение программы
Для анализа логики работы программы использовались инструменты пошагового выполнения кода.

3. Просмотр значений переменных
Помогает отслеживать изменения данных в процессе выполнения программы.

5. Запуск программы в режиме отладки
Программа запускалась в режиме Debug. Это позволило использовать все инструменты отладки и анализировать выполнение программы в реальном времени.
